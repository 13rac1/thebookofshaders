## Floor
Find the nearest integer less than or equal to the parameter

### Các phiên bản
```glsl
float floor(float x)  
vec2 floor(vec2 x)  
vec3 floor(vec3 x)  
vec4 floor(vec4 x)
```

### Các tham số
```x``` specify the value to evaluate.

### Mô tả
```floor()``` returns a value equal to the nearest integer that is less than or equal to ```x```.
