## Tan
Return the tangent of the parameter

### Các phiên bản
```glsl
float tan(float angle)  
vec2 tan(vec2 angle)  
vec3 tan(vec3 angle)  
vec4 tan(vec4 angle)
```

### Các tham số
```angle``` specify the quantity, in radians, of which to return the tangent.

### Mô tả
```tan()``` returns the trigonometric tangent of angle.
