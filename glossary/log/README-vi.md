## Log
Return the natural logarithm of the parameter

### Các phiên bản
```glsl
float log(float x)  
vec2 log(vec2 x)  
vec3 log(vec3 x)  
vec4 log(vec4 x)
```

### Các tham số
```x``` specify the value of which to take the natural logarithm.

### Mô tả
```log()``` returns the natural logarithm of ```x```.
